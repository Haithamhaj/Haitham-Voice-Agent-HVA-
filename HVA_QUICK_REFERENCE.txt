╔═══════════════════════════════════════════════════════════╗
║  🚀 التشغيل السريع                                        ║
╚═══════════════════════════════════════════════════════════╝

  1. انقر نقرتين على:  HVA Simple.command
  2. انتظر ظهور:      🎤 في شريط القوائم
  3. اضغط:            ⌘⇧H (Cmd+Shift+H)
  4. قل:              "هيثم" + أمرك
  5. شاهد:            النتيجة في نافذة نصية


╔═══════════════════════════════════════════════════════════╗
║  ⌨️  الاختصارات                                          ║
╚═══════════════════════════════════════════════════════════╝

  ⌘⇧H     →  بدء الاستماع
  ESC     →  إغلاق النافذة

╔═══════════════════════════════════════════════════════════╗
║  📝 أمثلة سريعة                                           ║
╚═══════════════════════════════════════════════════════════╝

  🧠 الذاكرة:
     هيثم، احفظ ملاحظة: اجتماع غداً
     هيثم، ابحث عن اجتماعات

  📧 Gmail:
     هيثم، اقرأ آخر إيميل
     هيثم، لخص آخر 5 إيميلات

  📁 الملفات:
     هيثم، اعرض الملفات في Downloads
     هيثم، افتح مجلد المستندات

  💻 النظام:
     هيثم، افتح Safari
     هيثم، افتح Chrome


╔═══════════════════════════════════════════════════════════╗
║  🎨 ألوان النافذة                                         ║
╚═══════════════════════════════════════════════════════════╝

  🎤 أزرق   →  أوامرك
  🤖 سماوي  →  ردود هيثم
  ✅ أخضر   →  نجاح
  ❌ أحمر   →  خطأ
  ℹ️  رمادي →  معلومات


╔═══════════════════════════════════════════════════════════╗
║  🔧 القائمة (انقر 🎤)                                     ║
╚═══════════════════════════════════════════════════════════╝

  🎤 Listen        →  بدء الاستماع
  📝 Show Window   →  إظهار النافذة
  🗑️  Clear History →  مسح السجل
  ℹ️  About         →  عن البرنامج
  ⏹️  Quit          →  إيقاف


╔═══════════════════════════════════════════════════════════╗
║  💡 نصائح                                                 ║
╚═══════════════════════════════════════════════════════════╝

  ✓ تحدث بوضوح وببطء معتدل
  ✓ ابدأ دائماً بـ "هيثم"
  ✓ استخدم جمل قصيرة ومباشرة
  ✓ تجنب الضوضاء في الخلفية
  ✓ اضغط 📌 Pin لإبقاء النافذة مفتوحة


╔═══════════════════════════════════════════════════════════╗
║  🔴 الإيقاف                                               ║
╚═══════════════════════════════════════════════════════════╝

  القائمة → ⏹️ Quit
  أو: Ctrl+C في Terminal
